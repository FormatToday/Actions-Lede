# Actions-OpenWrt
使用 Github Actions 每周一、三、五20点自动编译 OpenWrt 固件

源码仓库 https://github.com/coolsnowwolf/lede

编辑 .config 文件，把内容清空替换成自己的配置内容

点这右上角 ✰Star  变成 ★Unstar 即可开始编译

等待编译成功后，到Actions里下载固件即可
